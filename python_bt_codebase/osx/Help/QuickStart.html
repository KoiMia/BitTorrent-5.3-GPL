Quick Start

This application implements a file transfer protocol called BitTorrent. Each person downloading a set of files with BitTorrent also uploads parts of the same files to the other people downloading simultaneously. This relieves the burden on the originating server.

The person or web site offering the set of files for download is the the one who decides to use BitTorrent. BitTorrent does not have any search capabilities. Torrent files can be placed on a web site and sent through e-mail or instant message.

1 Download the torrent file. Torrent files are small files with names ending in ".torrent".
2 Open the torrent file either by double clicking it, dragging it onto the application icon or main window, or choosing File > Open.
The download should begin automatically and will be saved in your usual Internet download location.
3 Wait for the download to complete.

BitTorrent creates all the files immediately and fills them up with data as the downlnoad progresses. Please wait until the download is complete before using or moving the files.

When the download is complete, BitTorrent will continue to upload until the upload to downlwoad ratio equals 0.8, or whatever is set in the prefrences. Onces BitTorrent has finished uploading, it will begin downloading the first torrent in the list with a status equal to "Waiting." You may change the order of the queue by dragging rows.

If the download doesn't appear to be going anywhere, see Troubleshooting.
